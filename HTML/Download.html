Download

Here you can download RealTimeBattle in different formats. To install the program, follow the instructions in the INSTALL file!
  • The gzipped archive: Sourceforge: RealTimeBattle-#VERSION#.tar.gz Lyastor: RealTimeBattle-#VERSION#.tar.gz
  • The bzipped archive: Sourceforge: RealTimeBattle-#VERSION#.tar.bz2 Lysator: RealTimeBattle-#VERSION#.tar.bz2
  • We do not compile rpm:s any more. If anyone compile rpm:s and are prepared to maintain the rpm:s, please contact us.
  • Fredrik Hallenberg has put together a debian package.
  • It is possible to download RealTimeBattle from CVS. Learn about Sourceforge's CVS or go to the CVS-Help.
  • Old versions are available here:
    SourceForge: #SOURCEFORGEOLDVERSIONDIR#
    Lysator: #LYSATOROLDVERSIONDIR#

The documentation is made using sgml-tools which is capable to translate the source to a number of different formats. The html version of the documentation is available on the web. Archived versions of the documentation are downloadable here:
  • html
  • dvi
  • postscript
  • GNU-info
  • Text
  • SGML source
