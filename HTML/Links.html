Lysator Lysator is a computer association at Linköping University.
SourceForge Soureforge holds cvs, web-site, bug-tracking among other things for open-source projects. We have our cvs and bug-tracking system at sourceforge.
Gtk+ The GIMP Toolkit. This is a gui-package that is required to run the graphical version of RealTimeBattle.
Gtk+-Themes Here you can find themes to use with gtk+ >= 1.1.5
RobotBattle This is a the game that RealTimeBattle was inspired from. It is very nice, but only available for Windows and has an own programming-language, which is very limiting.
SGML-tools The package used to create the documentation for RealTimeBattle. It is able to convert the source to a diverse set of formats, like HTML, info, latex and ps.
Freshmeat The number one site for news on Linux applications. New versions of RealTimeBattle will be announced here.
Selim Issever Selim Issevers homepage with various robots and robot classes.
German RTB Documentation This is the homepage of the German translation of the RTB documentation. This page is maintained by Uwe Hermann.
Artur Adib This is the homepage of Artur Adib. One of the contributers to RealTimeBattle.
Jesús Arnáiz This is Jesús Arnáiz linuxpages with RealTimeBattle. He is working on a spanish translation of the RealTimeBattle docs. The RTB information is in the linux section. He has a spanish tutorial for writing robots.
Dieter Schwarz Dieter Schwarz has got a generic robot written in Tcl based on 'Rotate and Fire'
Jiro Sekiba, Minatsu Tukisima and Shinsuke Noguchi They have a site in Japanese with lots of information, e.g. robots to download and a mirror of the official homepage.
Guillaume Aubry A French site with pupose to gather French builders of robots.
Programming games links Dennis Lühring's large list to programming games and other ai stuff.
Linux4us They are hosting a weekly RTB tournament. The site is german but RTB related info are translated to english.
RobotWar for BeOS Cedric Neve and Christophe Belpaire are working on a robot game for BeOS, which will be compatible with rtb.
